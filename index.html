How AuraCode Works

1. Local Virtual Workspace

All files you see in the sidebar are held in a virtual, in-memory filesystem. You can add, edit, and delete files manually, or let the AI Agent do it automatically.

2. The Agent Workflow

When you ask the AI to perform a task, AuraCode constructs a detailed prompt including the request and your current directory files. The agent runs a structured cycle:

  • Analysis: Reads your request and scans active workspace files.
  • Planning: Formulates which files need editing or creation.
  • Execution: Outputs updates in structured formats that are automatically parsed and written to your workspace.

3. Live Preview & Exports

For web projects (HTML, CSS, JS), you can toggle the **Preview** tab. It spins up a sandboxed dynamic runner that simulates a local server and lets you interact with your code instantly. When you are ready, click **Export ZIP** to download the code to your computer.
